## 📖 Overview

This folder documents my HTML5 fundamentals as part of the [Full Web Development](../../README.md) repository — covering document structure, semantic markup, forms, and a first hands-on mini project.

---

## ✅ Topics Covered

- HTML document structure & boilerplate (`<!DOCTYPE>`, `html`, `head`, `body`)
- Comments and case-insensitivity of HTML
- Headings (`h1`–`h6`) and paragraphs
- Text formatting (bold, italic, underline, big/small)
- Horizontal rules, sub/superscript, `pre` tag
- Anchor tags & links, images
- Ordered, unordered, and description lists
- Tables (`tr`, `td`, `th`, `thead`, `tbody`, captions)
- Forms, input types, labels, buttons
- Semantic elements: `header`, `main`, `footer`, `section`, `article`, `aside`
- Non-semantic containers: `div`, `span`
- Class vs. ID
- `select` dropdowns, `iframe`, audio/video embedding

## 🚀 Mini Project — Personal Portfolio

A multi-page portfolio site built entirely with hand-written HTML, linking a home page to dedicated sub-pages:

```text
1-HTML/
├── 4-portfoilo_page.html     # Home page
├── 4.1-Education.html        # Education details
├── 4.2-Experience.html       # Experience & hackathons
├── 4.3-Projects.html         # Project write-ups
└── 4.4-About_section.html    # About me
```

This was the first project used to practice multi-page navigation and content structuring with plain HTML (no CSS styling applied on the sub-pages yet).
